# Team Images

Add your team member photos here with the following names:

- `member1.jpg` - Machine Learning Engineer
- `member2.jpg` - Full Stack Developer
- `member3.jpg` - Computer Vision Engineer
- `member4.jpg` - UI/UX Designer

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 400x400 pixels (square)
- **Aspect Ratio**: 1:1 (square images work best)
- **File Size**: Keep under 500KB for optimal loading

## How to Add Images

1. Place your team photos in this directory (`/public/images/team/`)
2. Name them according to the list above
3. The images will automatically appear in circular avatars on the Team page
